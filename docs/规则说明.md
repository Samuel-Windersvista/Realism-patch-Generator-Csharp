# SPT现实主义数值范围编辑生成器 v0.9 规则说明

本文档用于说明当前 rules 目录下几类规则文件的作用，以及 GUI 中各个分类的大致意义。

## 1. 规则文件位置

规则文件位于 rules 目录：

- rules/weapon_rules.json
- rules/attachment_rules.json
- rules/ammo_rules.json
- rules/gear_rules.json

GUI 不直接暴露整份 JSON，而是把这些文件拆成“分类 + profile + 字段范围”的形式供编辑。

## 2. 规则范围的基本结构

大多数可编辑节点本质上都是类似下面的结构：

```json
{
  "min": 0.0,
  "max": 0.0,
  "preferInt": false
}
```

含义：

- min：允许生成的下界
- max：允许生成的上界
- preferInt：生成时是否优先按整数处理

## 3. 武器规则

来自 weapon_rules.json，当前 GUI 中主要拆成以下分类：

- 武器全局夹紧
- 武器基础规则
- 武器口径补修
- 武器枪托补修

说明：

- 武器全局夹紧：用于限制最终值不能超出安全边界
- 武器基础规则：定义武器 profile 的主基准范围
- 武器口径补修：按口径族对基础规则做偏移修正
- 武器枪托补修：按枪托形态对后坐、操控等做补修

## 4. 附件规则

来自 attachment_rules.json，当前 GUI 中主要拆成以下分类：

- 附件全局夹紧
- 附件规则

说明：

- 附件全局夹紧：限制附件结果不能过强或过弱
- 附件规则：定义各类附件 profile 的基础范围

## 5. 弹药规则

来自 ammo_rules.json，当前 GUI 中主要拆成以下分类：

- 弹药基础规则
- 弹药特殊修正
- 弹药穿深档位修正

说明：

- 弹药基础规则：决定伤害、穿深、初速等主基准
- 弹药特殊修正：用于 AP、曳光、亚音速等特性的偏移
- 弹药穿深档位修正：按穿深等级做全局偏移补修

## 6. 装备规则

来自 gear_rules.json，当前 GUI 中主要拆成以下分类：

- 装备全局夹紧
- 装备规则

说明：

- 装备全局夹紧：限制移速惩罚、舒适度等结果范围
- 装备规则：定义护甲、头盔、胸挂、背包、耳机等 profile 的主基准

## 7. 字段理解建议

GUI 底部会为当前选中字段提供说明。实际调整时建议按以下思路：

- 先只改一个字段，便于观察效果
- 先缩小改动幅度，再逐步放大
- 优先改基础规则，再改补修规则
- 若出现极端结果，优先检查对应分类下的夹紧规则

## 8. 关于 profile 中文名

当前 GUI 已为常见 profile key 补充中文友好名称，例如：

- assault -> 突击步枪
- full_power_rifle_rimmed -> 全威力有底缘步枪弹
- intermediate_rifle_762x39 -> 7.62x39 中间威力步枪弹
- stockless -> 无枪托

这些名称主要服务于中文阅读与筛选理解，不改变底层规则 key。

## 9. 当前不做的事情

- GUI 不直接编辑整份规则 JSON
- GUI 不负责修改输入模板源文件
- GUI 不替代最终生成结果验证

规则编辑的目标是更清晰地调范围，而不是把所有底层结构都暴露给使用者。
